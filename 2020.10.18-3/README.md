# Решение

Решение представлено в файлах `./calc/**/*`, а для проверки, соответственно, в `./demo/**/*`.

Для запуска необходима платформа [nodejs](https://nodejs.org):

```bash
# расчёт параметров
npm run calc

# запуск демо в live-режиме
npm run demo:serve

# сборка демо в статические файлы
npm run demo:build
```
